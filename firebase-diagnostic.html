Common Issues & Solutions:

Issue 1: Realtime Database Not Enabled

Symptom: Error says "Database not initialized" or "database not found"

Solution:

  1. Go to Firebase Console - Realtime Database
  2. Click "Create Database"
  3. Choose your location
  4. Select "Start in test mode"
  5. Click "Enable"
  6. Wait 1-2 minutes

Issue 2: Permission Denied

Symptom: Error says "PERMISSION_DENIED"

Solution:

  1. Go to Firebase Console - Database Rules
  2. Replace rules with:
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
  1. Click "Publish"

Issue 3: Network/CORS Error

Symptom: Error says "CORS" or "network error"

Solution:

  1. Make sure you're accessing via http://localhost:8000
  2. Don't open HTML file directly (file://)
  3. Check internet connection

Quick Fix Steps:

Do these in order:
  1. Click "Run Full Diagnostic" above
  2. Read the error message carefully
  3. Follow the solution for your specific error
  4. Refresh this page and test again
