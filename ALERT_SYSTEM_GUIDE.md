# 🚨 Automatic Alert System Guide

## ✨ New Features

Your Diabetes Tracker now has a powerful automatic alert system that sends WhatsApp messages to multiple emergency contacts when:

1. **Low Sugar** - Below threshold (default: 70 mg/dL)
2. **High Sugar** - Above threshold (default: 250 mg/dL)
3. **Missed Entry** - No entry for X hours (default: 6 hours)

## 👥 Multiple Emergency Contacts

### Add Contacts
1. Go to **Settings** → **Emergency Contacts**
2. Enter contact name (e.g., "Dr. Smith")
3. Enter WhatsApp number with + (e.g., "+1234567890")
4. Select relationship (Doctor, Family, Friend, Caregiver)
5. Click **"Add Contact"**
6. Repeat for multiple contacts

### Manage Contacts
- **Call**: Click 📞 to call directly
- **Message**: Click 💬 to send WhatsApp message
- **Delete**: Click 🗑️ to remove contact

### Contact Types
- 👨‍⚕️ **Doctor** - Your primary physician
- 👨‍👩‍👧 **Family** - Family members
- 👥 **Friend** - Close friends
- 🤝 **Caregiver** - Professional caregiver

## 🚨 Automatic Alerts

### 1. Low Sugar Alert ⚠️

**Triggers when:**
- Sugar level drops below threshold
- Default: 70 mg/dL
- Customizable: 40-100 mg/dL

**What happens:**
1. WhatsApp opens for each emergency contact
2. Pre-filled message with:
   - Current sugar level
   - Threshold value
   - Time of reading
   - Urgent action needed
3. Local alert asks if you want to call
4. One-tap to call first contact

**Message Example:**
```
🚨 LOW SUGAR ALERT!

Sugar Level: 65 mg/dL
Threshold: 70 mg/dL

Time: 4/11/2024, 3:45 PM

Please check on the patient immediately!
```

### 2. High Sugar Alert 🚨

**Triggers when:**
- Sugar level exceeds threshold
- Default: 250 mg/dL
- Customizable: 200-400 mg/dL

**What happens:**
1. WhatsApp opens for each contact
2. Pre-filled warning message
3. Local alert notification
4. Recommends monitoring

**Message Example:**
```
⚠️ HIGH SUGAR ALERT!

Sugar Level: 280 mg/dL
Threshold: 250 mg/dL

Time: 4/11/2024, 8:30 PM

Please monitor the patient closely.
```

### 3. Missed Entry Alert ⏰

**Triggers when:**
- No entry logged for X hours
- Default: 6 hours
- Customizable: 2-24 hours
- Sends once per day

**What happens:**
1. Checks every hour
2. If no entry for set duration
3. Sends alert to all contacts
4. Shows last entry details

**Message Example:**
```
⏰ MISSED ENTRY ALERT!

No sugar level logged for 7 hours.

Last Entry: 4/11/2024 1:00 PM
Sugar: 125 mg/dL

Please check on the patient.
```

## ⚙️ Configure Alerts

### Low Sugar Alert
1. Go to **Settings** → **Automatic Alerts**
2. Toggle **Low Sugar Alert** ON/OFF
3. Set threshold (40-100 mg/dL)
4. Default: 70 mg/dL

### High Sugar Alert
1. Toggle **High Sugar Alert** ON/OFF
2. Set threshold (200-400 mg/dL)
3. Default: 250 mg/dL

### Missed Entry Alert
1. Toggle **Missed Entry Alert** ON/OFF
2. Set hours (2-24 hours)
3. Default: 6 hours

## 📱 How It Works

### When You Add Entry:
```
1. Enter sugar level
2. Click "Save Entry"
3. App checks thresholds
4. If low/high → Alert triggered
5. WhatsApp opens for each contact
6. Send messages
```

### Missed Entry Check:
```
1. Runs every hour
2. Checks last entry time
3. If > X hours → Alert triggered
4. Sends once per day
5. Resets next day
```

- ✅ Inform contacts they're listed

### Threshold Settings

- ✅ Not too sensitive (avoid false alarms)

- ✅ Set based on your schedule

## 🔔 Alert Frequency

### Low/High Sugar
- Triggers: Every time threshold crossed
- Frequency: Immediate on each entry
- Limit: None (every occurrence)

### Missed Entry
- Triggers: After X hours
- Frequency: Once per day
- Limit: Won't spam contacts

## 🛠️ Troubleshooting

### Alerts not sending
**Check:**
- [ ] Contacts added with +
- [ ] Alert toggles are ON
- [ ] Thresholds set correctly
- [ ] WhatsApp installed
- [ ] Internet connection

### Too many alerts
**Solution:**
- Adjust thresholds
- Increase missed entry hours
- Remove some contacts
- Disable specific alerts

### Contacts not receiving
**Solution:**
- Verify phone numbers
- Check country codes
- Test with "Message" button
- Confirm WhatsApp active

## 📞 Emergency Response

### If You Receive Alert:

2. Ask if they're okay
3. Remind to eat something
4. Stay on phone if needed
5. Call 911 if no response

**High Sugar Alert:**
1. Call to check in
2. Ask about symptoms
3. Remind to take insulin
4. Monitor next few hours
5. Suggest doctor call

**Missed Entry Alert:**
1. Call or text patient
2. Ask if everything okay
3. Remind to log entry
4. Check if help needed
5. Visit if no response
