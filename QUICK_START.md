# 🚀 Quick Start Guide

## 📱 WhatsApp Auto-Send Setup

### Step 1: Enter Your WhatsApp Number
1. Open the app
2. Go to **Settings** tab
3. Find **WhatsApp Auto-Send** section
4. Enter your number with country code
   - Example: `+1234567890` (USA)
   - Example: `+919876543210` (India)
   - Example: `+447123456789` (UK)

### Step 2: Set Send Time
1. Choose when you want daily reports
2. Default is 9:00 PM
3. Pick a time when you're usually free

### Step 3: Enable Auto-Send
1. Toggle **Enable Auto-Send** to ON
2. You'll get a confirmation message
3. Reports will be sent automatically every day

### Step 4: Test It
1. Click **Send Now to WhatsApp**
2. WhatsApp will open with your daily report
3. Send it to yourself or your doctor

## ⏰ Daily Reminders Setup

### Step 1: Set Your Times
1. Go to **Settings** → **Daily Reminders**
2. Set 3 reminder times:
   - **Morning**: When you wake up (e.g., 8:00 AM)
   - **Afternoon**: After lunch (e.g., 2:00 PM)
   - **Evening**: Before bed (e.g., 8:00 PM)

### Step 2: Enable Reminders
1. Toggle **Enable Reminders** to ON
2. Browser will ask for notification permission
3. Click **Allow**

### Step 3: Get Alerts
- You'll receive 3 notifications per day
- Each reminder says: "Time to check your blood sugar!"
- Click the notification to open the app

## 👨‍⚕️ Emergency Contact Setup

### Step 1: Add Contact
1. Go to **Settings** → **Emergency Contact**
2. Enter contact name (e.g., "Dr. Smith" or "Mom")
3. Enter phone number with country code
4. Click **Save Contact**

### Step 2: Use in Emergency
1. If you feel unwell or have extreme readings
2. Go to Settings
3. Click **Call Emergency Contact**
4. Confirm the call
5. Your phone will dial immediately

## 🆘 Troubleshooting

### WhatsApp not opening?
- Check your number has country code (+)
- Make sure WhatsApp is installed
- Try clicking "Send Now" manually first

### Reminders not working?
- Check notification permissions in browser
- Make sure reminders are enabled (toggle ON)
- Verify times are set correctly
- Keep the app tab open in background

### Emergency call not working?
- Verify phone number is saved
- Check number format (+1234567890)
- Make sure your device can make calls
- Test with a regular call first

## 📱 Browser Permissions

The app needs these permissions:

1. **Notifications** - For daily reminders
   - Chrome: Settings → Privacy → Notifications
   - Safari: Preferences → Websites → Notifications

2. **Phone Calls** - For emergency contact
   - Automatically works on mobile devices
   - Desktop will open phone app if available
